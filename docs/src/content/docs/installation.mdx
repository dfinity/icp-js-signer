## Install

<Tabs>
  <TabItem label="npm">```bash npm install @icp-sdk/signer ```</TabItem>
  <TabItem label="pnpm">```bash pnpm add @icp-sdk/signer ```</TabItem>
  <TabItem label="yarn">```bash yarn add @icp-sdk/signer ```</TabItem>
</Tabs>

## Peer Dependencies

This package requires `@icp-sdk/core` as a peer dependency:

<Tabs>
  <TabItem label="npm">```bash npm install @icp-sdk/core ```</TabItem>
  <TabItem label="pnpm">```bash pnpm add @icp-sdk/core ```</TabItem>
  <TabItem label="yarn">```bash yarn add @icp-sdk/core ```</TabItem>
</Tabs>
